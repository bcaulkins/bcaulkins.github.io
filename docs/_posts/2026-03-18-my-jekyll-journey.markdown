I was bored yesterday and I started looking into the old blog services of the early 2000s like LiveJournal. I used to have a LiveJournal that I wrote to semi-regularly and it was a fun creative outlet. No one saw it, but I enjoyed it. It was my "public" journal of sorts. I think George R. R. Martin used to have a LiveJournal account as well, though I'm sure his was far more entertaining as mine.

I digress...

Where was I? Yes, the old bloggling platforms of yesterday. I didn't find anything I liked, LiveJournal is now owned by Russia. Wordpress is overkill, Blogger was OK, but I just didn't like it for whatever reason. Actually, I don't know why I said that. I think I liked Blogger back in the day. Maybe I should have two, one here on github and a mirror on Blogger? I don't know, seems complicated. I will come back to Blogger and give it a better review.

I saw Ghost was an option, which sounded cool, but I wasn't looking forward to self hosting something. Not that I'm against self hosting, I have a jellyfin setup, Plex, pihole, my own Foundry TTRPG setup running on my NAS, so I'm no stranger to self hosting. What I wanted when I came back to blogging was a relatively simple setup that just worked, but was still technical to be a fun challenge. And that's where I landed on Jekyll.

There was a link on Jekyll's site that mentioned using it with github pages to host a static website and blog. That sounded perfect. I struggle with using git, and having a system that forces me to learn how to use source control, from the terminal no less, to update a silly blog sounded like a neat project.

I spent an hour or more going through the setup process, created a new master branch, got a local copy of my repo cloned to my PC then started playing around. It felt a lot like the days of playing in this new sandbox, learning html and css, building silly and weird web pages. So, I think this will serve has my home for the ramblings of a tech noob, someone who knows just enough to be dangerous, but not quite enough to get paid.

So far this will be a place that I can throw random thoughts, but I also run another website, an actual site that has a real vision. It's called TheWinnerIs.org. The site needs to be fixed. It needs some TLC and I need to expand it to use more advanced technology to be a better user experience. I hope to use this space as a way to explore my frustrations and my wins.

There will be more posts in between as well, some serious, some not so serious, but all entertaining in some fashion, if nothing more than to be a source of embarassment. I am quite self deprecating.

I will see you in the next post, until then have a wonderful day.

Brian
